To install the system, download it, and then run make build. Then, 
one will have the option to run make clean and make build.

The system can be tested by running make test, which will run our test suite. 

We used the following packages: ounit2, ANSITerminal

To interact with the GUI, user must have XQuartz and graphics installed. Run 
make graphics in the terminal and an XQuartz application will open with 
instructions on how to use it.
